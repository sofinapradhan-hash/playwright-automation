This repository contains Playwright-based end-to-end tests and supporting page objects and utilities for https://www.saucedemo.com/.

Quick start

- Install dependencies: `npm ci`
- Install Playwright browsers: `npx playwright install`
- Run tests: `npm test` (alias for `npx playwright test`)

Notes
- Replace `<OWNER>/<REPO>` in the badge URL above with the repository owner and name to enable the CI badge.
- Tests are configured to upload Playwright reports and test results in CI.
